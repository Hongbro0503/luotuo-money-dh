# 老舍 『駱駝祥子』 돈(錢) 서사 — 디지털인문학 마크업·시각화

전남대학교 디지털인문학 / 홍승현

老舍의 장편소설 『駱駝祥子(낙타샹즈)』 **1~10장**에 나타나는 돈 관련 표현을 전수 마크업하고,
**"오가는 돈(사건) ≠ 말해지는 돈(담론)"** 이라는 문제의식으로 시각화한 작업물입니다.

🔗 **라이브: https://hongbro0503.github.io/luotuo-money-dh/**

---

## 무엇을 한 작업인가

소설 속 '돈'은 두 얼굴을 가집니다. 실제로 **오가는 돈**(빼앗기고, 벌고, 사는 돈 = 사건)과,
인물들이 입으로 **말하는 돈**(이자·시세·"돈이 돈을 낳는다" 같은 담론)이 그것입니다.
이 둘을 같은 '돈'으로 뭉뚱그리지 않고 태그 속성으로 갈라서, 한 작품 안에서
돈이 **언제 흐르고 언제 말해지는지**를 측정 가능한 데이터로 만들었습니다.

- 본문의 돈 표현마다 `<돈>` 태그를 달아 **성격·방향·유형·액수·주체·관여**를 기록
- 그 태그를 표(CSV)로 펼쳐 차트·관계망·서사 곡선으로 시각화
- 결과: 8장이 돈을 가장 많이 **말하면서도** 정작 오가는 돈은 적다는 식의 비대칭이 드러남

## `<돈>` 태그 스키마 (3층)

| 층위 | 속성 | 값 예시 |
|------|------|---------|
| **측정층** (본문에 적힌 것) | 유형 · 액수 · 단위 | `搶`·`掙`·`利錢` / `96`·`1分` / `塊` |
| **해석층** (연구자 판단) | 성격 · 방향 | 흐름·값·계산·흥정·목표 / 획득·상실 |
| **식별** | 주체 · 상대 · 관여 | `#xiangzi`·`#gaoma` / 직접 ● · 간접 ○ |

성격이 핵심 축입니다 — **흐름**=사건(돈이 실제로 오감), **값·계산·흥정**=담론(돈을 말함), **목표**=지향.

## 주요 수치 (1~10장)

| 수치 | 의미 |
|------|------|
| **291** | 돈 태그 (한·중 양측 정렬 582) |
| **1,807** | 정렬 문장 (한·중 평행코퍼스, 협업 구축) |
| **147 : 113** | 사건 : 담론 (오간 돈 ↔ 말해진 돈) |
| **67 : 1** | 노동 : 비노동 (수입원의 압도적 비대칭) |

## 데이터

- 마크업 원본: 1~10장 합본 XML (`<돈>` 태그 582개, 한·중 양측 정렬)
- 분석용 표: 돈 토큰 291개를 한 행씩 펼친 CSV (성격·방향·유형·액수·단위·주체·관여·범주)
- 모든 차트는 이 CSV 한 장에서 생성 — index.html 안에서 두 파일 모두 내려받을 수 있습니다.
